Introduction

JavaScript is a powerful programming language used for web development.

It can be run in the browser and on the server (Node.js).

console.log("Hello World");

JavaScript is versatile and used everywhere.

  • Client-side scripting
  • Server-side development

It's used to build interactive UIs.

JavaScript Syntax

Syntax defines how we write JavaScript code.

let x = 5;

It uses curly braces to define blocks.

if (x > 3) { console.log(x); }
Variables

Variables store data.

var name = "Alchie";

They can be declared using var, let, or const.

  • var - function scoped
  • let - block scoped
  • const - block scoped and constant
Functions

Functions are reusable blocks of code.

function greet() { return "Hi Pandit ji!"; }

They can take parameters and return values.

Loops

Loops help repeat tasks easily.

for (let i = 0; i < 5; i++) { console.log(i); }

Different types include for, while, do-while.

  • for loop
  • while loop
  • do-while loop
